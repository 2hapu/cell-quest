# 셀 퀘스트 : 오피스 몬스터 도감

엑셀 스프레드시트로 위장한 텍스트 수집형 RPG.

## 플레이
GitHub Pages 주소로 접속하면 바로 플레이할 수 있습니다.
- 셀 클릭 또는 숫자키(1~9)로 진행
- **ESC** : 가짜 매출 시트로 즉시 위장 전환
- 저장: 메뉴의 **백업 코드**를 발급받아 보관 → 다음 접속 시 붙여넣어 복원

## 게임 내용
- 37종 오피스 몬스터 도감 (포획 / 레벨업 진화 / 황금 희귀 개체)
- 5개 층 + 보스 트레이너 5인 + 엔딩 후 야근차원 & 전설 포획
- 상점, 보관함, 타입 상성 (카페인 > 서류 > 전자 > 카페인 / 야근)
